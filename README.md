# AppInventorJapaneseDocsについて

---
このプロジェクトはAppInventorのドキュメントを日本語に翻訳するプロジェクトです。

---
## 翻訳に協力するには
Githubに会員登録を行い、日本App Inventorユーザー会のGoogleグループの
翻訳のディスカッションで協力したい旨をお申し出下さい。
https://groups.google.com/d/topic/app-inventor-jp/V90gaeUIduE/discussion

しばらく待つと編集者として登録されます。

その間に作業に必要なGitのクライアントをインストールします。

WindowsならTortoiseGitが良いかもしれません。
http://tortoisegitjapan.com/

Macなら
http://mac.github.com/

そもそもgitの使い方がわからない方はこちら
http://sourceforge.jp/magazine/09/02/02/0655246
http://blog.katty.in/758
http://blog.designrecipe.jp/2009/7/15/git-basic/

## 翻訳作業の流れ
1. 担当者が設定されていないISSUEの中から、翻訳したいページのISSUEを探します。
2. 翻訳するページのISSUEの担当者を自分に設定します。
3. ページを翻訳します。
4. 翻訳が終わったらページをpushします。  （コミットコメントに 「#1 closed」とか入れると一緒にISSUEも閉じてくれるようです。）
以上の繰り返しです。

※チケットは同時に複数取っても良いですが、あまり作業が進まない状態で取りっぱなしはご遠慮ください。  ※翻訳は中途半端な状態でギブアップしてもかまいません。途中までのソースでpushしていただき、その際はISSUEを閉じずに、自分を担当者から外して下さい。

---
## 誤訳を見つけた場合
ISSUEを新規に作成してください。  ISSUEには対象ファイル名、修正箇所、修正内容を書いてください。  翻訳者は誤訳のISSUE対応を行う場合は通常の翻訳作業の流れでISSUEを処理してください。

※翻訳者自身が誤訳を見つけた場合でも翻訳のISSUEを閉じている場合は、ISSUEをあげるところから行ってください。
