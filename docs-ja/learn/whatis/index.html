App Inventor とは

App Inventor は、Web ブラウザを使って Android 端末向けのアプリケーションを開発できる仕組みです。 開発したアプリケーションは、コンピュータに接続した携帯電話でも、エミュレータの上でも動作します。 また、開発したアプリケーションは、App Inventor サーバに蓄積され、あなたのプロジェクトがどこかに行ってしまわないようにしてくれます。

アプリケーションの開発は、以下のツールを使って行います :

  • The App Inventor Designer あなたの作るアプリケーションに使う部品を選択する場所です。
  • App Inventor Blocks Editor 「プログラムブロック」を組み立てて、App Inventor Designer で選択した部品がどのように動くかを決める場所です。 パズルのピースを組み立てるようにプログラムブロックを組み立てることで、視覚的にプログラムを組みたてていきます。

プログラムブロックを組むごとに、携帯電話上で起動しているあなたのアプリケーションは形を変えていきます。プログラムを組むとすぐにあなたのアプリケーションをテストすることができるのです。 プログラムの作成が完了したら、あなたのアプリケーションを「パッケージ」にして、携帯電話にインストールすることで App Inventor から離れても実行できるスタンドアロンのアプリケーションにすることができます。

もし Android 携帯電話をお持ちでない場合は、 Android エミュレータ を使って開発することもできます。 このソフトウェアはあなたのコンピュータの上で実行され、Android 携帯電話のようにふるまいます。

App Inventor 開発環境は、Mac OS X, GNU/Linux, Windows をサポートしており、いくつかのメジャーな Android 携帯電話のモデルを想定した環境を提供します。 App Inventor で開発したアプリケーションは、どんな Android 携帯電話にでもインストールする事ができます。 （ システム要件 も参照してください）

App Inventor を使う前に、 コンピュータの準備をする を読み、 App Inventor Setup パッケージをお使いのコンピュータにインストールしてください。
