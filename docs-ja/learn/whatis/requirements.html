システム要件

App Inventor を利用するためのシステム要件は以下のとおりです。

コンピューターとオペレーティングシステム

  • Macintosh (Intel プロセッサ搭載のもの): Mac OS X 10.5 または 10.6
  • Windows: Windows XP, Windows Vista, Windows 7 を搭載したコンピューター
  • GNU/Linux: Ubuntu 8 以降, Debian 5 以降を搭載したコンピューター

ブラウザー

  • Mozilla Firefox 3.6 以降
  • Apple Safari 5.0 以降
  • Google Chrome 4.0 以降
  • Microsoft Internet Explorer 7 以降

携帯電話

App Inventor で作成したアプリケーションは Android を搭載した携帯電話上で動作します。 また、以下のリストにある携帯電話については、開発環境とセットアップソフトウェアをインストールするだけで開発を行うことができます。

  • Google: Nexus S, Nexus One, Android Dev Phone 1, Android Dev Phone 2, Ion
  • T-Mobile: G1*, MyTouch 3G*
  • Motorola: Droid*

このリストに無い携帯電話については、携帯電話の製造元が提供する Windows 用のドライバーが必要になります。** どの携帯電話を使う場合でも、携帯電話に SD カードを搭載しておいてください。SD カードがないと、App Inventor を使って開発を行うことができません。

App Inventor を使った開発には、リストに挙げた端末の他、HTC, Samsung, Dell などから発売されている多数の Android 携帯電話を利用する事ができます。 ただし、多くの場合、製造元から追加のソフトウェアをダウンロードし、インストールする必要が生じます。 そのような作業が必要になる携帯電話のうち、いくつかについての詳細な情報を 携帯電話をセットアップする のページの下部にある表に載せてあります。 また、お使いの携帯電話がその表になくても、App Inventor を使った開発に利用できることが多いです。 App Inventor User Forum *** をのぞいてみて、"Getting set up and connecting your phone" のセクションにあるトピックの中から、同じ携帯電話をお使いのユーザーからのアドバイスを探してみてください。

* 他のキャリアが販売している同等の端末も含みます
** 訳注: 原文では特に "Windows driver" をインストールするように、と書かれている箇所です。 App Inventor でアプリケーションを開発するとき、お使いのコンピューターに USB ケーブルで Android 携帯電話を接続する必要がありますが、Windows では、USB 接続を行うために専用のドライバーをインストールする必要があり、また携帯電話の製造元が提供しているドライバーを別途インストールしなければならないケースがほとんどのため、Windows を特別扱いしているものと思われます。
Macintosh は、Android の接続にドライバーのインストールは不要です。
Linux の場合は、設定ファイルを手動で編集する必要があります。詳しくは、Google の提供しているドキュメントを参照してください（注：英語の情報になります）
*** 訳注: 日本語の情報源としては、日本App Inventorユーザー会の Google グループがあります。困ったらここをのぞいてみて既にあるトピックを探してみたり、質問したりしてください。マナーを大切に。
