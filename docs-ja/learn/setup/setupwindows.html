Windows 用の App Inventor Setup ソフトウェアをインストールする

Windows 用の App Inventor Setup ソフトウェアのインストールは、ふたつの手順からなります。
  1. App Inventor Setup ソフトウェアパッケージをインストールする。この手順は、すべての Android 端末で同じで、また、Windows XP・Vista・7 で同じです。
  2. あなたの Android 携帯電話用の Windows ドライバーをインストールする

App Inventor Setup ソフトウェアパッケージをインストールする

あなたが管理者権限を持つアカウントからインストールを実行することをお勧めします。 これは、マシンすべてのユーザーのためのソフトウェアをインストールします。 あなたが管理者権限を持っていない場合、インストールはまだ動作するはずですが、 App Inventor は、あなたがインストールするときに使用したアカウントからのみ使用できます。
  1. インストーラーを ダウンロード します。
  2. あなたのダウンロードファイルやデスクトップの中から AppInventor_Setup_Installer_v_1_2.exe (~92 MB) ファイルを探します。 お使いのコンピューター上でのダウンロードの場所は、お使いのブラウザーの設定によって異なります。
  3. ファイルを開きます。
  4. インストーラーの手順をクリックします。 インストール場所を変更しないでください。 ただし、ドライバーを確認するため必要とする場合があるため、インストールディレクトリーを記録します。 ディレクトリーは、お使いの Windows のバージョンとあなたが管理者としてログインしているかどうかによって異なります。

Setup ソフトウェアの場所を確認

ほとんどのケースでは、App Inventor は、Setup ソフトウェアを見つけることができるはずです。 しかし、ソフトウェアの場所を要求した場合、入力するパスは C:\Program Files\Appinventor\commands-for-Appinventor です。64ビットのマシンを使用している場合は、Program Files ではなく、Program Files (x86) を入力する必要があります。 また、あなたが管理者としてソフトウェアをインストールしなかった場合、C:\Program Files ではなくあなたのローカルディレクトリーにインストールされました。 あなたはソフトウェアが正しいパス名がわかるようインストール場所を検索する必要があります。

携帯電話のドライバー

App Inventor Setup ソフトウェアは、これらの一般的な Android 携帯電話用のドライバーを含んでいます。

  • T-Mobile G1* / ADP1
  • T-Mobile myTouch 3G* / Google Ion / ADP2
  • Verizon Droid (not Droid X)
  • Nexus One
  • Nexus S

ここに記載されていない携帯電話は、 手動でドライバーの入手とインストール を行う必要があります。

いくつかのケースでは、Windows 用のドライバーは常に自動的にインストールされないので、 その場合は、 手動でドライバーのインストール を行う必要があります。
何か正しく動作しませんか？ トラブルシューティングのページ にアクセスしてください。または、 App Inventor ユーザーフォーラム を手助けのためにチェックしてください。

次の手順

あなたの Android 携帯電話をセットアップしてください。 その次に 携帯電話であなたの初めてのアプリを組み立ててください。
