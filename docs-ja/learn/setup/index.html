App Inventor Setup

App Inventor は web ブラウザー上で動作します。
携帯電話をお持ちの場合は、携帯電話を USB ケーブルでコンピューターにつなぎ、作業する事ができます。作業を進めるのに従い、作成したアプリケーションがだんだんと携帯電話の上にあらわれてきます。
携帯電話をお持ちでない場合は、App Inventor のパッケージに含まれてる携帯電話のエミュレーターを使って作業することができます。
アプリケーションの組み立てが終わったら、出来上がったアプリケーションを「パッケージ」し、「アプリケーションパッケージ」（つまり、Android の apk ファイル）を作り、他の Android アプリケーションと同じように、アプリケーションを共有したり、Android 携帯電話にインストールしたりできるようにします。

アプリケーション開発を始めるために、以下の手順を完了する必要があります

  • 手順 1: コンピューターをセットアップする

  • 手順 2: Android 携帯電話を使うか、エミュレーターを使うかにより、以下のどちらかの手順を完了してください。
    • Android 携帯電話をセットアップする 携帯電話上で、最初のアプリケーションを組み立てる もしくは
    • エミュレーター上で、最初のアプリケーションを組み立てる

コンピューターをセットアップする

システム要件:

コンピューターとOS

  • Macintosh (Intel プロセッサー搭載のもの): Mac OS X 10.5, 10.6*
  • Windows: Windows XP, Windows Vista, Windows 7
  • GNU/Linux: Ubuntu 8 以降, Debian 5 以降

ブラウザー

  • Mozilla Firefox 3.6 以降
    • Note: Firefox に NoScript 拡張機能をインストールしてお使いの場合、この拡張機能を無効にする必要があります。 トラブルシューティングのページの該当項目 もご覧ください。
  • Apple Safari 5.0 以降
  • Google Chrome 4.0 以降
  • Microsoft Internet Explorer 7 以降

Java 設定を確認する

お使いのコンピューターで Java 6 (Java 1.6 と呼ばれる場合もあります) が動作する必要があります。
Java は www.java.com からダウンロード出来ます。

Java の設定を、以下のテストを両方行って確認してください。

  1. Java test page を web ブラウザーで開いてください。
    Java が動作していて、かつバージョンが Java 1.6 であることを示すメッセージが表示されることを確認してください。
  2. App Inventor の Java テストを このリンク を開いて実施してください。
    このテストは、お使いのブラウザーが Java が動作するように設定されているか、および、Java web Start でアプリケーションを起動できるかを確認するものです。
テストが失敗した場合、お使いのコンピューターで App Inventor は動作しません。
問題が解決するまで、App Inventor を使って開発はしないようにしてください。

App Inventor Setup ソフトウェアをインストールする

App Inventor を使う前に、お使いのコンピューターにソフトウェアをインストールする必要があります。
必要なソフトウェアは App Inventor Setup というパッケージにまとめられて提供されています。
お使いの OS に合わせたインストール手順に従い、インストールを行ってください。
インストールが終わったら、携帯電話、あるいはエミュレーター上で、デモアプリケーション (Hello Purr) を組み立てる手順に移りましょう。

  • Mac OS X での手順
  • GNU/Linux での手順
  • Windows での手順
うまくいきませんか？ トラブルシューティング や、 App Inventor User Forum ** で、解決方法を探してみてください。
