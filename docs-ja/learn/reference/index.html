部品

以下のページでは、App Inventor でアプリケーションを作るときに利用可能な部品について説明しています。

それぞれの部品にはメソッド、イベント、プロパティを持たせる事ができます。
多くのプロパティはアプリケーション側から変更することができます。こうしたプロパティは値を読み出す、あるいは書き込む事ができるブロックを持っています。
一方、アプリケーション側から変更できないプロパティもあります。こうしたプロパティは値を読み出す事だけが出来るブロックを持っており、値を書き込むことはできません。
このリファレンスでは、読み出し専用のプロパティの名前は 斜体文字 で書かれています。
２、３のプロパティだけは、Designer の中でだけ利用可能になっています。

  • 基本部品
  • メディア部品
  • アニメーション部品
  • ソーシャル部品
  • センサー部品
  • レイアウト部品
  • LEGO® MINDSTORMS® 部品
  • その他の部品
  • 準備中の部品

LEGO と MINDSTORMS は LEGO グループの登録商標です。
