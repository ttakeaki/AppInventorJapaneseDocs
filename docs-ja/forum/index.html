App Inventor for Android ユーザーディスカッションフォーラム

フォーラムに入る前に、まず下記の 重要な覚え書き をお読みください。フォーラムとうまく付き合っていきましょう。
  • サブフォーラムには いくつでも 入る事ができます。サブフォーラムのタイトルをクリックして詳細を確認し、気になるフォーラムに入会しましょう。 フォーラムのティップス を読んで、これらのフォーラムの役割についてもっと知りましょう。
  • よくある質問 と トラブルシューティング のページ、それから App Inventor の ドキュメント は、 質問をする前に読んでおきましょう。 質問に対して回答がもらいやすくなります。
  • フォーラムの過去ログを 検索 しましょう。過去に同じ問題にあたった人がいて、既に回答が得られているかもしれません。
  • フォーラムに バグを報告しないでください。 バグや機能のリクエストは ISSUES LIST に登録してください。App Inventor 開発チームは ISSUES LIST で、開発の優先順位付けを公開しています。*
  • 質問の件名は簡潔にまとめ、詳細な情報を本文に書きましょう。 フォーラムの参加者に問題を理解してもらえるようにしてください。アプリケーションコードの断片、スクリーンショット、ソースファイルは有用な情報となります。

*ISSUES LIST は英語の情報になります。バグや機能のリクエスト一覧、その優先順位付けの情報は、リンク先からさらに "Click here to search the existing issues" というリンクをたどった先で読めます。
