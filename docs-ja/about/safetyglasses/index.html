これは「ベータ版」です！


Google Labs で公開されていたものと同様、ベータ版の App Inventor にもこなれていない箇所や、現在利用できない機能があります。
そのため、いくつかの場面でご面倒をおかけすることがあります。例えば…
  • Windows 用のデバイスドライバーをインストールしなければならない
  • Java をコンピューターにインストールしなければならない
  • つまらない携帯電話の設定をしなければならない
…など。
こういった面倒をかけなくても済むよう、鋭意開発を進めているところです。
ソフトウェアがベータ版であるにもかかわらず、App Inventor を使ってくださる皆さんに感謝いたします。

決まり文句のような文章に見えるかもしれませんが、心の底から、本当に、感謝しているのです。
皆さんが App Inventor を使ってくださることで、私たちはこの先 App Inventor をもっと良いものにしていくことができるのです！

ともに良いものを作っていきましょう！
